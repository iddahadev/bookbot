# Bookbot

This is my first Python project. It is part of the curiculum on [boot.dev](https://www.boot.dev) to become a Backend Developer!

## Run Locally

Clone the project

```bash
git clone https://github.com/iddahadev/bookbot.git
```

Go to the project directory

```bash
cd bookbot
```

Run the script

```bash
python3 main.py
```
